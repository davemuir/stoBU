Class RadarSeriesStyles
 
Packageeng.series
Classpublic class RadarSeries

The RadarSeries class (<RadarSeries>) is the series of the RadarChart node (<RadarChart>).

The layout is as follows:

 <KoolChart backgroundColor="0xFFFFEE" cornerRadius="12" borderStyle="solid">
        <RadarChart isSeriesOnAxis="false" type="polygon" paddingTop="20" paddingBottom="20">
             <radialAxisRenderers>
               <AxisRenderer horizontal="true" visible="true"/>
               <AxisRenderer horizontal="false" visible="false"/>
              </radialAxisRenderers>
              <radialAxis>
               <LinearAxis/>
              </radialAxis>
              <angularAxis>
                  <CategoryAxis id="aAxis" categoryField="catName" displayName="Category" />
              </angularAxis>
              <series>
                  <RadarSeries field="year2005" radius="5" displayName="2005"/>
                  <RadarSeries field="year2006" radius="5" displayName="2006"/>
                  <RadarSeries field="year2007" radius="5" displayName="2007"/>
              </series>
   </RadarChart>
 </KoolChart>
