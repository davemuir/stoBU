alwayShowLabels
Type: Boolean

Indicates whether or not to display the label when its size is smaller than that of the series area.
If alwayShowLabels="true", then the label will be always displayed. If alwayShowLabels="false" and there is no enough space, then the label will not be displayed.

The default value is true.		Matrix2DSeries

fill
Type: Object
Sets the fill color to be used to fill items by the item renderer. 		Matrix2DSeries
  
fillJsFunction
Type: Object
Sets the JavaScript function to be called to fill colors generated by the user-defined logic.

Sets the value of the fillJsFunction property to the name of the JavaScript function.

  e.g. <Line2DSeries field="Profit" fillJsFunction="fillJsFunc">

The signature of fillJsFunction is as follows:

  function_name(index, values) { ... }
index: The item index in the series.
values: The item value. The value is an array and each element in the array is different depending on the chart type.
  • Bar2D(3D)Series (Bar2DSeries, Bar3DSeries) 0: X-axis value, 1: Y-axis value
  • Column2D(3D)Series (Column2DSeries, Column3DSeries) 0: X-axis value, 1: Y-axis value
  • Area2DSeries (Area2DSeries) 0: X-axis value, 1: Y-axis value
  • Bubble3DSeries (Bubble3DSeries) 0: X-axis value, 1: Y-axis value 2: radius
  • Line2DSeries (Line2DSeries) 0: X-axis value, 1: Y-axis value
  • Pie2D(3D)Series (Pie2DSeries, Pie3DSeries) 0: value, 1: percentage, 2: nameField
  • If the minField is defined in From-To charts, the last index of values is minField.
  • Matrix2DSeries (Matrix2DSeries) 0: X-axis value, 1: Y-axis value, 2: Z-axis value
Matrix2DSeries
  
fills
Type: Array
Sets the array of fill color objects used by each item renderer.
If the number of array elements is less than the number of items, then the colors defined in the array will be repeatedly used to fill items. e.g. if the number of items in the series is 10 and the number of fill color objects is 5, then each of 5 colors will be used to fill the first 5 items and the 5 colors will be used again to fill the other items. 
    <Area2DSeries ... >
     <fills>
      <SolidColor color="0xCC66FF"/>
      <SolidColor color="0x9966CC"/>
      <SolidColor color="0x9999CC"/>
     </fills>
    </Area2DSeries>
Matrix2DSeries

insideLabelJsFunction
Type: Object
Sets the JavaScript function to be called to generate the user-defined label. 
  e.g. <Matrix2DSeries ... insideLabelJsFunction="labelJsFunc"/>

The signature of insideLabelJsFunction is as follows:

  function_name(seriesId, index, data, values){...}
seriesId: Series ID index: The item index in the series. data: The record of the dataset declared for creating the chart. values: The item value. The value is an array and each element in the array is different depending on the chart type.
  • Bar2D(3D)Series (Bar2DSeries, Bar3DSeries) 0: X-axis value, 1: Y-axis value
  • Column2D(3D)Series (Column2DSeries, Column3DSeries) 0: X-axis value, 1: Y-axis value
  • Area2DSeries (Area2DSeries) 0: X-axis value, 1: Y-axis value
  • Bubble3DSeries (Bubble3DSeries) 0: X-axis value, 1: Y-axis value 2: radius
  • Line2DSeries (Line2DSeries) 0: X-axis value, 1: Y-axis value
  • Pie2D(3D)Series (Pie2DSeries, Pie3DSeries) 0: value, 1: percentage, 2: nameField
  • If the minField is defined in From-To charts, the last index of values is minField.
  • Matrix2DSeries (Matrix2DSeries) 0: X-axis value, 1: Y-axis value, 2: Z-axis value
Matrix2DSeries
