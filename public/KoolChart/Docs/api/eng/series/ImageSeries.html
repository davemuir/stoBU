Packageeng.series
Classpublic class ImageSeries

The ImageSeries class (<ImageSeries>) is the series of the ImageChart node (<ImageChart>). Set the value of the imgSource property to the array of ImageSourceItem. The valid values of the imageDisplayType property are as follows:
  • "single": Represents data by using a single image. maintainAspectRatio="true": Uses the same aspect ratio for items. If the extra space remains unused, the space will be filled with a bar maintainAspectRatio="false": The image size is adjusted automatically (mostly vertical).
  • "singleRepeat": Represents data by using a single image. Uses the same aspect ratio for items. If the extra space remains unused, the space will be filled with the same image.
  • "multiple": Represents data by using multiple images. Each image has a unique value. The size of an image is the unit value. The image which has the largest unit is displayed at the bottom and the next largest one is displayed on the top of the largest one.
The default style of itemRenderer is the ImageItemRenderer class.

<ImageSeries> inherits from the parent classes.

   <KoolChart backgroundColor="0xFFFFFF"  cornerRadius="12" borderStyle="solid">
   <Options>
     <Caption text="Image Chart" />
     <SubCaption text="(2009 ~ 2012)" textAlign="center"/>
    </Options>
    <NumberFormatter id="numFmt" /> 
    <ImageChart id="chart" showDataTips="true" gutterLeft="20" gutterRight="20"  showLabelVertically="true">
     <horizontalAxis>
      <CategoryAxis id="hAxis" categoryField="Year"/>
    </horizontalAxis>
     <verticalAxis>
      <LinearAxis id="vAxis"/>
     </verticalAxis>
    <series>
     <ImageSeries yField="Data1" imageDisplayType="single" styleName="seriesStyle" displayName="Male" formatter="{numFmt}">
      <imgSource>
               <ImageSourceItem maintainAspectRatio="true" url="../Samples/Images/1-1-1.png"/>
               </imgSource>
               <showDataEffect>
                <SeriesSlide duration="1000" direction="up"/>
               </showDataEffect>
     </ImageSeries>
     <ImageSeries yField="Data2" imageDisplayType="single" styleName="seriesStyle2" displayName="Female" formatter="{numFmt}">
      <imgSource>
                <ImageSourceItem maintainAspectRatio="true" url="../Samples/Images/1-1-2.png"/>
              </imgSource>
              <showDataEffect>
            <SeriesSlide duration="1000" direction="up"/>
            </showDataEffect>
    </ImageSeries>
   </series>
   <verticalAxisRenderers>
    <Axis2DRenderer axis="{vAxis}" visible="false" includeInLayout="false"/>
   </verticalAxisRenderers>
  </ImageChart>
  <Style>
   .seriesStyle
   {
    labelPosition:outside;
    fontFamily:Myriad;
    color:0x0066FF;
    fontSize:11;
   }
   .seriesStyle2
   {
    labelPosition:outside;
    fontFamily:Myriad;
    color:0xFF66FF;
    fontSize:11;
   }
  </Style>
  lt;/KoolChart>

alwayShowLabels
Type: Boolean

Indicates whether or not to display the label when its size is smaller than that of the series area.
If alwayShowLabels="true", then the label will be always displayed. If alwayShowLabels="false" and there is no enough space, then the label will not be displayed.

The default value is true.		ImageSeries
  
columnWidthRatio
Type: Number
Sets the column width as the percentage of the width of the category. The valid range is 0~1.
If the value is 1, then the whole area will be used to draw bars. If the value is 0.6, then 60% of the area will be used to draw bars.

The actual width of the bar is the minimum value of columnWidthRatio and maxColumnWidth

The default value is 0.65.		ImageSeries
  
fillJsFunction
Type: Object
Sets the JavaScript function to be called to fill colors generated by the user-defined logic. If this property is set, fill and fills will be ignored but if the returned value is null, then fills or fill will be applied. 		ImageSeries
  
fills
Type: Array
Sets the array of fill color objects used by each item renderer. If the number of array elements is less than the number of items, the colors defined in the array will be repeatedly used to fill colors in items. e.g. If the number of items in the series is 10 and the number of fill color objects is 5, then each of 5 colors will be used to fill the first 5 items and the 5 colors will be used again to fill the other items. 
     <ImageSeries ... >
      <fills>
       <SolidColor color="0xCC66FF"/>
       <SolidColor color="0x9966CC"/>
       <SolidColor color="0x9999CC"/>
      </fills>
     </ImageSeries>
