Classpublic class Pie3DSeries

The Pie3DSeries class (<Pie3DSeries>) is the series of the Pie3DChart node (<Pie3DChart>).
The Pie3DSeries class (<Pie3DSeries>) represents data as a pie with slices.

The layout is as follows:

 <KoolChart backgroundColor="0xFFFFEE"  cornerRadius="12" borderStyle="solid">
   <Options>
    <Caption text="Annual Report"/>
  </Options>
  <Pie3DChart showDataTips="true">
    <series>
     <Pie3DSeries field="Profit"/>
    </series>
   </Pie3DChart>
  </KoolChart>

fillJsFunction
Type: String
Sets the JavaScript function to be called to fill colors generated by the user-defined logic.

Sets the value of the fillJsFunction property to the name of the JavaScript function.

 e.g. <Plot2DSeries field="Profit" fillJsFunction="fillJsFunc">

The signature of fillJsFunction is as follows:

 function_name(index, values) { ... }
index: The item index in the series.
values: The item value. The value is an array and each element in the array is different depending on the chart type.
  • Bar2D(3D)Series (Bar2DSeries, Bar3DSeries) 0: X-axis value, 1: Y-axis value
  • Column2D(3D)Series (Column2DSeries, Column3DSeries) 0: X-axis value, 1: Y-axis value
  • Area2DSeries (Area2DSeries) 0: X-axis value, 1: Y-axis value
  • Bubble3DSeries (Bubble3DSeries) 0: X-axis value, 1: Y-axis value 2: radius
  • Line2DSeries (Line2DSeries) 0: X-axis value, 1: Y-axis value
  • Pie2D(3D)Series (Pie2DSeries, Pie3DSeries) 0: value, 1: percentage, 2: nameField
  • If the minField is defined in From-To charts, the last index of values is minField.
  • Matrix2DSeries (Matrix2DSeries) 0: X-axis value, 1: Y-axis value, 2: Z-axis value
Pie3DSeries
  
fills
Type: Object
Sets the array of fill color objects used by each item renderer. If the number of array elements is less than the number of items, the colors defined in the array will be repeatedly used to fill colors in items. e.g. If the number of items in the series is 10 and the number of fill color objects is 5, then each of 5 colors will be used to fill the first 5 items and the 5 colors will be used again to fill the other items. 
    <Plot2DSeries ... >
     <fills>
      <SolidColor color="0xCC66FF"/>
      <SolidColor color="0x9966CC"/>
      <SolidColor color="0x9999CC"/>
     </fills>
    </Plot2DSeries>

  • Bubble3DSeries (Bubble3DSeries) 0: X-axis value, 1: Y-axis value 2: radius
  • Line2DSeries (Line2DSeries) 0: X-axis value, 1: Y-axis value
  • Plot2DSeries (Plot2DSeries) 0: X-axis value, 1: Y-axis value
  • Pie2D(3D)Series (Pie2DSeries, Pie3DSeries) 0: value, 1: percentage, 2: nameField
  • If the minField is defined in From-To charts, the last index of values is minField.

labelPosition
Type: String
Sets the position of numeric labels. The valid values are as follows:
  • "none" - Do not display labels.
  • "outside" - Display labels outside slices.
  • "callout" - Display labels outside slices and draw the line that connects the label and the slice.
  • "inside" - Display labels inside the slice.
  • "insideWithCallout" - Displays the value inside the chart. If the font size of labels is smaller than insideLabelSizeLimit, then the value will be displayed outside the chart with the line (callout).
The default value is "none".		Pie3DSeries
