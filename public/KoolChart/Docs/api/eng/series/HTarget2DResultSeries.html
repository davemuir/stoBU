Classpublic class HTarget2DResultSeries

The HTarget2DResultSeries class (<HTarget2DResultSeries>) is used to draw the actual values of the Target vs Actual chart (Combination2DChart). You can create a Target vs Actual chart by setting the series of Combination2DChart to this class. The HTarget2DGoalSeries class (<HTarget2DGoalSeries>) is used to draw the target values of the Target vs Actual chart and the HTarget2DResultSeries class (actual) should be defined first. The default item renderer is BoxItemRenderer and the default value of the barWidthRatio property is 0.8. The default fill style is the SolidColor instance and the hexadecimal value of the color is "값은0x299DE5". The default value of the stroke style is Stroke(0x000000, 1, 1).

The layout is as follows:

   <Combination2DChart>
     <series>
    Actual
    <HTarget2DResultSeries />
  
    Target
    <HTarget2DGoalSeries />
     </series>
   </Combination2DChart>

<Combination2DChart> inherits from the parent classes.

fillJsFunction
Type: String
Sets the JavaScript function to be called to fill colors generated by the user-defined logic.

Sets the value of the fillJsFunction property to the name of the JavaScript function.

  e.g. <Column2DSeries field="Profit" fillJsFunction="fillJsFunc">

The signature of fillJsFunction is as follows:

  function_name(index, values) { ... }
index: The item index in the series.
values: The item value. The value is an array and each element in the array is different depending on the chart type.
  • Bar2D(3D)Series (Bar2DSeries, Bar3DSeries) 0: X-axis value, 1: Y-axis value
  • Column2D(3D)Series (Column2DSeries, Column3DSeries) 0: X-axis value, 1: Y-axis value
  • Area2DSeries (Area2DSeries) 0: X-axis value, 1: Y-axis value
  • Bubble3DSeries (Bubble3DSeries) 0: X-axis value, 1: Y-axis value 2: radius
  • Line2DSeries (Line2DSeries) 0: X-axis value, 1: Y-axis value
  • Pie2D(3D)Series (Pie2DSeries, Pie3DSeries) 0: value, 1: percentage, 2: nameField
  • If the minField is defined in From-To charts, the last index of values is minField.
  • Matrix2DSeries (Matrix2DSeries) 0: X-axis value, 1: Y-axis value, 2: Z-axis value
  • CandleSeries (Candlestick2DSeries) 0:X-axis value, 1:openField value, 2:closeField value, 3:highField value, 4:lowField value
