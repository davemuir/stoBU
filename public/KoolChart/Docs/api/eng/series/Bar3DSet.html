Class Bar3DSetStyles
 
Packageeng.series
Classpublic class Bar3DSet

The Bar3DSet class (<Bar3DSet>) contains multiple Bar3DSeries, which operates as a single series.

The layout is as follows:

 <Combination3DChart showDataTips="true">
   <verticalAxis>
    <CategoryAxis categoryField="Month"/>
      </verticalAxis>
      <series>
         <Bar3DSet type="clustered">
           <series>
      <Bar3DSeries xField="Cost" displayName="Cost"/>
      <Bar3DSeries xField="Revenue" displayName="Revenue"/>
       </series>
      </Bar3DSet>
    <Line2DSeries xField="Revenue" displayName="Revenue"/>
   </series>
  </Combination3DChart>

type
Type: String
Sets the type of the representation when Bar3DSet contains multiple Bar3DSeries.
The valid values are as follows:
  • "clustered": The data elements of each series are displayed as they are in the order.
  • "overlaid": The data elements of each series are displayed overlaid. The lastly defined series is displayed on the top.
  • "stacked": The data elements of each series are displayed as the horizontal stack.
  • "100%": The data elements of each series are displayed as the horizontal stack by calculating the percentage of the horizontal space.
The default value is "clustered".		Bar3DSet
