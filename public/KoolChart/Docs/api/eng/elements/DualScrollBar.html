Class DualScrollBarStyles
 
Packageeng.elements
Classpublic class DualScrollBar

DualScrollBar needs to be set in dataSelector when the number of data is large in DualChart.

The configuration of DualScrollBar is as follows:

  ...
   <mainChart>
    ...
   </mainChart>
   <subChart>
    ...
   </subChart>
   <dataSelector>
    <DualScrollBar visibleItemSize="30" inverted="false"/>
   </dataSelector>
  ...
