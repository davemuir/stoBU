Class CrossRangeZoomerStyles
 
Packageeng.elements
Classpublic class CrossRangeZoomer

By using CrossRangeZoomer, you can display crosshairs in charts and zoom in on certain areas.

The layout is as follows:

The CrossHair class (<CrossHair>) inherits from the parent classes and has the following additional properties.

   <KoolChart backgroundColor="0xFFFFFF" borderStyle="solid" cornerRadius="12">
  ...
  <Line2DChart showDataTips="true" selectionMode="single">
   ...
   <annotationElements>
    <CrossRangeZoomer enableZooming="false" horizontalLabelFormatter="{numFmt}"/>
   </annotationElements>
  </Line2DChart>
  lt;/KoolChart>
   </pre>
