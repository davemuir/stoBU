Packageeng.elements
Classpublic class BrokenAxisLine

The BrokenAxisLine class is to draw the wave pattern for the broken axis.
You can change the shape of the wave-patterned area, the line color and the fill color. 
  <annotaionElements>
   <BrokenAxisLine brokenAxisAreaColor="0xffffff" flatRatio="0">
    <brokenAxisLineStroke>
     <Stroke color="0xff0000" weight="1" alpha="1"/>
    </brokenAxisLineStroke>
   </BrokenAxisLine>
  </BrokenAxisLine>
