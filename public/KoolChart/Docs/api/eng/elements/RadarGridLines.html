Class RadarGridLinesStyles
 
Packageeng.elements
Classpublic class RadarGridLines

By using RadarGridLines, you can draw grid lines in the background of the chart. The types of the line are radial, angular and both. radial: Draws the line between the center and the axis (AngularAxis). angular: Draws the line between each of the labels in the axis. The instance of ChartGridLines (<ChartGridLines>) should be set to the value of the backgroundElements property.

The RadarGridLines class (<RadarGridLines>) inherits from the parent classes and has the following additional properties.

 <KoolChart backgroundColor="0xFFFFFF" cornerRadius="12" borderStyle="solid">
 ...
 <RadarChart id="chart1" isSeriesOnAxis="true" type="polygon" paddingTop="20" paddingBottom="20" showDataTips="true">
  ...
  <backgroundElements>
   <RadarGridLines direction='both' radarChangeCount='1' radarFill='0xEFFFFF' radarAlternateFill='0xFFFFFC'> <!-- The colors of radarFill and radarAlternateFill will be used alternately.-->
     <radialStroke> <!-- Defines the radial line -->
     <Stroke color='0x000000' alpha='1' weight='1'/>
    </radialStroke>
     <angularStroke> <!-- Defines the angular line -->
     <Stroke color='0xFF0000' alpha='1' weight='1'/>
    </angularStroke>
    </RadarGridLines>
  </backgroundElements>
 ...
 </RadarChart>
 </KoolChart>
