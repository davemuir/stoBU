Packageeng.elements
Classpublic class Box

The Box class (<Box>) can contain the Label object (<Label>) or the Image object (<Image>).
The objects contained in the CanvasElement class (<CanvasElement>) are represented by their X, Y coordinates but the objects contained in the Box class (<Box>) are represented by their created order.
