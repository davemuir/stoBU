Classpublic class AxisRange

By using AxisRange, you can adjust the color and label of the axis range.

The AxisRange class (<AxisRange>) inherits from the parent classes and has the following additional properties.

The layout is as follows:

 <KoolChart backgroundColor="0xFFFFFF" borderStyle="solid" cornerRadius="12">
 ...
 <Line2DChart showDataTips="true" fontFamily="Myriad" selectionMode="single">
  ...
  <backgroundElements>
   <GridLines/>
   <AxisMarker>
                <ranges>
     <AxisRange startValue="1000" endValue="2000" label="Area 1" fontSize="11" labelHorizontalAlign="left" color="0xFF00FF">
      <fill>
       <SolidColor color="0x00FF99" alpha="0.2"/>
      </fill>
     </AxisRange>
     <AxisRange startValue="Mar" endValue="May" label="Area 2" fontSize="11" labelVerticalAlign="bottom" color="0x0066FF" horizontal="false">
      <fill>
       <SolidColor color="0x0099FF" alpha="0.2"/>
      </fill>
     </AxisRange>
     <AxisRange startValue="Jul" endValue="Nov" label="Area 3" fontSize="11" labelVerticalAlign="bottom" color="0x0066FF" horizontal="false">
      <fill>
       <SolidColor color="0xdd99FF" alpha="0.2"/>
      </fill>
     </AxisRange>
    </ranges>
   </AxisMarker>
  </backgroundElements>
 </Line2DChart>
 </KoolChart>
