Class WingSeriesSlideStyles
 
Packageeng.effects
Classpublic class WingSeriesSlide

The WingSeriesSlide effect is the animation effect for Wing charts, which offers the sliding effect while data rendering. 

  
   <showDataEffect>
    <WingSeriesSlide/>
   <showDataEffect>


Styles
 StyleDescriptionDefined by
  
direction
Type: String

Sets the slide direction. The valid values are "left","right","up" and "down".

In the case of Column2DWing chart, "up" is same to "left" and "down" is same to "right". In the case of Bar2DWing chart, "left" is same to "up", "right" is same to "down". The default value is "left".		WingSeriesSlide
  
duration
Type: Number

Sets the time required to complete the effect. The unit is millisecond.
The default value is 500.

WingSeriesSlide
  
elementOffset
Type: Number

Sets the time between the animation effect of the first item and that of the next item.

If you set as elementOffset="0", the animation for all items start and end at the same time.
The default value is 20.		WingSeriesSlide
  
id
Type: String

Sets id. id is an unique identifier defined by users.

You can access to an object by using id.
WingSeriesSlide
  
minimumElementDuration
Type: Number

Sets the minimum time required to complete the effect. The unit is millisecond.
If the value of "minimumElementDuration" is larger than the value of "duration", "duration" will be ignored.

The default value is 0.		WingSeriesSlide
  
offset
Type: Number

Sets the delay time of the animation effect. The unit is millisecond.
It is the time gap between two series when multiple series are defined.

The default value is 0.		WingSeriesSlide
