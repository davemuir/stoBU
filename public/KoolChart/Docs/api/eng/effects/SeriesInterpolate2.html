Class SeriesInterpolate2Styles
 
Packageeng.effects
Classpublic class SeriesInterpolate2

The SeriesInterpolate2 effect gradually moves the data points from the center of Y-axis to the actual values when the Line2DSeries class (<Line2DSeries>) is used.

The SeriesInterpolate2 effect can be used only when the Line2DSeries is used.

The layout is as follows:

  <KoolChart>
   ...
    <Line2DSeries ... >
     <showDataEffect>
      <SeriesInterpolate2 duration="600"/>
     </showDataEffect>
    </Line2DSeries>
   ...
  </KoolChart>
