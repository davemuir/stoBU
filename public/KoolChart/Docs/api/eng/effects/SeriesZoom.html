Class SeriesZoomStyles
 
Packageeng.effects
Classpublic class SeriesZoom

The SeriesZoom effect starts at the reference point in the chart and gradually expands.

The layout is as follows:

  <KoolChart>
   ...
    <Line2DSeries ... >
     <showDataEffect>
      <SeriesZoom duration="600" horizontalFocus="center" verticalFocus="center/>
     </showDataEffect>
    </Line2DSeries>
   ...
  </KoolChart>

relativeTo
Type: String
Sets the reference point for the focus of the zoom. The valid values are series" and "chart".
If you set as relativeTo="series", the reference point will be the series item.
If you set as relativeTo="chart", the reference point will be the center of chart area.
The default value is series.		SeriesZoom
  
verticalFocus
Type: String
