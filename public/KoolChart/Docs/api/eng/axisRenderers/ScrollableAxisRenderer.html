Packageeng.axisRenderers
Classpublic class ScrollableAxisRenderer

The ScrollableAxisRenderer class (<ScrollableAxisRenderer>) is used to create scrollable charts.
If you want to add the scroll function to the chart, then you need to set an axis renderer to the instance of ScrollableAxisRenderer.
e.g. if you want to add the scroll function to the column 2D chart, then you need to assign CategoryLinearAxis to the horizontal axis and set its horizontalAxisRenderers to the instance of ScrollableAxisRenderer.

The layout is as follows:

  <KoolChart backgroundColor="0xFFFFEE"  cornerRadius="12" borderStyle="solid">
    <Column2DChart showDataTips="true" gutterRight="10">
     <series>
      <Column2DSeries id="cs1" yField="Data1" displayName="Data1"/>
     </series>
     <horizontalAxis>
      CategoryLinearAxis id="hAxis" categoryField="Gu"/>
     </horizontalAxis>
     <horizontalAxisRenderers>
      <ScrollableAxisRenderer axis="{hAxis}" visibleItemSize="10" scrollSensitivity="4"/>
     </horizontalAxisRenderers>
    </Column2DChart>
  </KoolChart>
