Package eng.axisRenderersClasses
 

The axisRenderers package contains the classes for the Kool Chart Axis 2D, 3D Renderer classes


Classes
 ClassDescription
 AngularAxisRenderer The AngularAxisRenderer class (<AngularAxisRenderer>) is used to define the rendering method of the category for Radar charts.
 Axis2DRenderer The Axis2DRenderer class (<Axis2DRenderer>) is a rendering method for the axis of 2D charts.
 Axis2DWingRenderer The Axis2DWingRenderer class (<Axis2DWingRenderer>) is a rendering method for the axis of Column2DWing chart and Bar2DWing chart.
 Axis3DRenderer The Axis3DRenderer class (<Axis3DRenderer>) is a rendering method for the axis of 3D charts.
 BrokenAxis2DRenderer The BrokenAxis2DRenderer class (<BrokenAxis2DRenderer>) is a rendering method for the broken axis of 2D charts.
 BrokenAxis3DRenderer The BrokenAxis2DRenderer class (<BrokenAxis2DRenderer>) is a rendering method for the broken axis of 3D charts.
 ScrollableAxisRenderer The ScrollableAxisRenderer class (<ScrollableAxisRenderer>) is used to create scrollable charts.
