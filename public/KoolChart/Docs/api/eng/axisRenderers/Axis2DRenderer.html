Packageeng.axisRenderers
Classpublic class Axis2DRenderer

The Axis2DRenderer class (<Axis2DRenderer>) is a rendering method for the axis of 2D charts.
You can adjust the thickness and color of the axis, the location of ticks, the color of ticks, etc.
The corresponding axis must be defined in the axis node (e.g. <LinearAxis>).

The layout is as follows:

 <Axis2DRenderer axis="{vAxis}" tickLength="20" minorTickLength="0" tickPlacement="outside" placement="left"
        canDropLabels="false" showLabels="true" labelAlign="center">
     <axisStroke>
      <Stroke weight="10" color="0x66CCFF" />
     </axisStroke>
     <tickStroke>
      <Stroke weight="2" color="0xFF0000" />
    </tickStroke>
     <minorTickStroke>
      <Stroke weight="1" color="0x0000FF" caps="square"/>
    </minorTickStroke>
  </Axis2DRenderer>

canDropLabels
Type: Boolean
Indicates whether or not to delete some labels to force all the labels into the layout neatly along the axis.

If the value is true, then the overlapped labels will be deleted.
The labels are layered out at the regular interval and some labels can be skipped if necessary.

If the value is false, then the labels will be rotated or zoomed in/out. (if internal fonts are used.)
The default value is dependent on the axis type.

Axis2DRenderer

It can be used to minimize the space for displaying labels.

The default value is true. It displays labels up and down alternately.

Type: String
Sets the location where minor ticks are placed.
You can use the following options.
  • "inside" - Draws minor ticks inside data area.
  • "outside" - Draws minor ticks inside label area.
  • "cross" - Draws minor ticks on the axis.
  • "none" - Does not draw minor ticks.

placement
Type: String
Sets the location of the axis in charts.
The valid values for the vertical axis are left and right. The valid values for the horizontal axis are top and bottom.
For the vertical axis, the default location of the first axis is left and for the horizontal axis, the default location of the first axis is bottom.
For Cartesian charts, the second axis is automatically located at the opposite of the first axis.
e.g. if you set as placement="right" for the first vertical axis, then the second vertical axis will be located at the left side of the chart. 		Axis2DRenderer
