Classpublic class AngularAxisRenderer

The AngularAxisRenderer class (<AngularAxisRenderer>) is used to define the rendering method of the category for Radar charts.
You can adjust the thickness and color of the axis, the location of ticks, the color of ticks, etc.
The corresponding axis must be defined in the axis node (e.g. <CategoryAxis>).

The layout is as follows:

  <RadarChart>
    <angularAxis>
     <CategoryAxis id="aAxis" categoryField="catName" displayName="Category"/>
    </angularAxis>
    <angularAxisRenderers>
     <AngularAxisRenderer axis="{aAxis}" styleName="hangingCategoryAxis"/>
    </angularAxisRenderers>
    ...
    ...
    ...
  </RadarChart>
