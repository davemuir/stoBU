Class VLinearGaugeStyles
 
Packageeng.charts
Classpublic class VLinearGauge

The VLinear class (<VLinear>) represents the target data and the actual data as the vertical bar. The outer bar represents the target value and the inner bar represents the actual value.

<VLinear> has the following features.

  • The outer bar represents the target value and the inner bar represents the actual value.
  • You can change the target and actual values.
  • You can change the color of the inner bar.
  • The actual value can be displayed as text and the location, font and style of the text can be changed.

The layout is as follows: 

  <KoolChart backgroundColor="0xffffff" cornerRadius="12" horizontalAlign="center" verticalAlign="middle" borderStyle="solid">
  <VLinear 
   width="150"
   height="400"
   minimum="0"
   maximum="200"
   labels="[0,20,40,60,80,100,120,140,160,180,200]"
   tickInterval="20"
   value="160"
   targetMark="120"
   snapInterval="1"
   valueLabelJsFunction="labelFunc"
   valueLabelStyleName="valueLabel"
  />
  <Style>
   .valueLabel{
    fontSize:11;
    fontWeight:bold;
   }
  </Style>
 </KoolChart>
