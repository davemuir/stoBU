Class Bar3DChartStyles
 
Packageeng.charts
Classpublic class Bar3DChart

Bar3D charts represent data with cubes and their lengths proportional to the values
The presenting types of Bar3D charts are as follows:
  • simple columns
  • clustered columns
  • stacked
  • 100% stacked

One or more Bar3DSeries have to be assigned to the series node.

<Bar3DChart> has the following additional properties compared to <Bar2DChart>.
  • cubeDepth: The cube depth
  • cubeAngleRatio: The percentage of the cube angle.

The layout is as follows:

 <KoolChart backgroundColor="0xFFFFEE"  cornerRadius="12" borderStyle="solid">
   <Options>
    <Caption text="Profit/Cost/Revenue (Monthly)"/>
  </Options>
  <Bar3DChart showDataTips="true">
    <verticalAxis>
     <CategoryAxis categoryField="Month" />
    </verticalAxis>
    <series>
     <Bar3DSeries xField="Profit"/>
     <Bar3DSeries xField="Cost" />
     <Bar3DSeries xField="Revenue" />
    </series>
   </Bar3DChart>
  </KoolChart>

Type: String
Sets the JavaScript function to be called to display the user-defined tooltips when mouse over items.
Sets the value of the dataTipJsFunction property to the name of the JavaScript function as follows: e.g.)

The signature of dataTipJsFunction is as follows: 

 function_name(seriesId, seriesName, index, xName, yName, data, values) { ... }
seriesId: Series ID
seriesName: The series name, which is set as the value of the displayName property.
xName: The value of the displayName property in the horizontalAxis node.
yName: The value of the displayName property in the verticalAxis node.
data: The record of the dataset declared for creating the chart.
values: The item value. The value is an array and each element in the array is different depending on the chart type.
  • Bar2D(3D)Series (Bar2DSeries, Bar3DSeries) 0: X-axis value, 1: Y-axis value
  • Column2D(3D)Series (Column2DSeries, Column3DSeries) 0: X-axis value, 1: Y-axis value
  • Area2DSeries (Area2DSeries) 0: X-axis value, 1: Y-axis value
  • Bubble3DSeries (Bubble3DSeries) 0: X-axis value, 1: Y-axis value 2: radius
  • Line2DSeries (Line2DSeries) 0: X-axis value, 1: Y-axis value
  • Pie2D(3D)Series (Pie2DSeries, Pie3DSeries) 0: value, 1: percentage, 2:nameField
The data string returned by the JavaScript function is the content of tooltips. 		Bar3DChart

type
Type: String
Sets the representation type of Bar3D charts.
The valid values are as follows:
  • "clustered": The data elements of each series are displayed as they are in the order.
  • "overlaid": The data elements of each series are displayed overlaid. The lastly defined series is displayed on the top.
  • "stacked": The data elements of each series are displayed as the horizontal stack.
  • "100%": The data elements of each series are displayed as the horizontal stack by calculating the percentage of the horizontal space.
The default value is "clustered".		Bar3DChart
