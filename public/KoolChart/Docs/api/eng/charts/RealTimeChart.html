Class RealTimeChartStyles
 
Packageeng.charts
Classpublic class RealTimeChart

Real-time charts represent data based on the number of data or the time period. Unlike other charts (data is refreshed when the dataProvider property is set.), the real-time chart attaches new data to the end of the previous data. e.g. if you set as dataDisplayType="dataSize" and displayDataSize="10", then the latest 10 items will be displayed in the chart regardless of the number data. If you set as dataDisplayType="time", then you have to set the property of the corresponding axis to the instance of the DateTimeAxis class (<DateTimeAxis>). If you set as interval="10" and displayDataSize="60", then the chart will display data received within the last 60 seconds with the 10 seconds interval. The actual number of items to be displayed in the chart is 6.

The layout is as follows:

    <KoolChart backgroundColor="0xFFFFFF" cornerRadius="12" borderStyle="solid">
   <Options>
    <Caption text="Stock Monitoring" />
    <SubCaption text="Real Time Chart" fontSize="11" textAlign="right"/>
    <Legend/>
   </Options>
    <DateFormatter id="dateFmt" formatString="HH:NN:SS" /> 
   <RealTimeChart id="chart" dataDisplayType="time"  timePeriod="60" interval="3" showDataTips="true">
    <backgroundElements>
     <GridLines direction="both"/>
    </backgroundElements> 
    <horizontalAxis>
       <DateTimeAxis id="hAxis" dataUnits="seconds" labelUnits="seconds" dataInterval="3" interval="3" formatter="{dateFmt}" displayLocalTime="true"/>
    </horizontalAxis>
    <series>
     <Column2DSeries labelPosition="outside" yField="Volume" xField="Time" 
      displayName="Trading Volume"
      itemRenderer="GradientColumnItemRenderer">
       <filters>
          <DropShadowFilter distance="3" color="0x666666" alpha=".8"/>
       </filters>
       <fill>
       <SolidColor color="0xB0C759"/>
       </fill>
      <verticalAxis>
         <LinearAxis id="vAxis1"  title="Volume" minimum="0" maximum="10000"/>
      </verticalAxis> 
     </Column2DSeries>
     <Line2DSeries labelPosition="up" xField="Time" yField="Price" displayName="Stock Price" interpolateValues="false"
        itemRenderer="mx.charts.renderers.CircleItemRenderer"  radius="5" fill="0xFFFF00">
      <filters>
       <DropShadowFilter distance="3" color="0x666666" alpha="0.8"/>
      </filters>
      <stroke>
       <Stroke color="0xE48701" weight="2"/>
      </stroke>
      <lineStroke>
        <Stroke color="0xE48701" weight="4"/>
       </lineStroke>
        <verticalAxis>
       <LinearAxis id="vAxis2" title="Price" minimum="0" maximum="1000"/>           
      </verticalAxis>
     </Line2DSeries>
    </series>
    <verticalAxisRenderers>
     <Axis2DRenderer placement="left" axis="{vAxis1}">
     </Axis2DRenderer>
     <Axis2DRenderer placement="right" axis="{vAxis2}">
     </Axis2DRenderer>
      </verticalAxisRenderers>
   </RealTimeChart>
   <HttpServiceRepeater url="http://www.koolchart.com/realtimeSample/data4.php" target="{chart}" interval="3"  method="get"/>
  </KoolChart>

 StyleDescriptionDefined by
  
dataDisplayType
Type: String
Sets the display type. You can define the number of data to be displayed in the chart or the time interval during which the number of data received will be displayed in the chart. If you set as dataDisplayType="time", the number of data received during the time interval will be displayed in the chart. The default is dataSize. 		RealTimeChart
  
displayCompleteCallFunction
Type: String

Sets the JavaScript function to be called when the chart rendering is completed.

The chart ID will be passed as parameter. 		RealTimeChart
  
enablePattern
Type: String

Sets patterns for visually impaired users.

The string, "usePattern=true", must be added to chartVars in the HTML file.
and the value of the patternMode property must be "true" in Column2DChart. 		RealTimeChart
  
interval
Type: Number
Sets the time interval when dataDisplayType="time". The unit is second. 		RealTimeChart
  
timePeriod
Type: Number
Sets the time period when dataDisplayType="time". It is the time period during which data is displayed in the chart. e.g. if you set as timePeriod="60", the data received within 1 minute before the last data is received will be displayed in the chart. The unit is second. 		RealTimeChart
  
touchMoveEnable
Type: Boolean

Indicates whether or not to create the touchMove event in the browser
when users move a certain area of the chart with a finger.
If the touchMove event is created in the browser
then the chart can be scrolled when the touchMove event occurs
otherwise the chart cannot be scrolled when the touchMove event occurs.

The default value is true.		RealTimeChart
