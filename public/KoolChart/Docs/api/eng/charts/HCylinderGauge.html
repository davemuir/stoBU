Classpublic class HCylinderGauge

The HCylinder class (<HCylinder>) represents the target and actual data as the horizontal cylinder. The outer cylinder represents the target value and the inner cylinder represents the actual value. The HCylinder class (<HCylinder>) inherits from the VCylinder class (<VCylinder>) and all the features of <VCylinder> can be used in <HCylinder>.

<HCylinder> has the following features.

  • The outer cylinder represents the target value and the inner cylinder represents the actual value.
  • You can change the target and actual values.
  • You can change the color of the inner cylinder.
  • The actual value can be displayed as text and the location, font and style of the text can be changed.

The layout is as follows: 

  <KoolChart backgroundColor="0xFFFFFF" paddingTop="50" cornerRadius="12" horizontalAlign="center" 
    verticalScrollPolicy="off" verticalAlign="middle" borderStyle="solid">
  <HCylinder
   width="400"
   height="150"
   minimum="0"
   maximum="200"
   labels="[0,20,40,60,80,100,120,140,160,180,200]"
   tickInterval="20"
   value="160"
   targetMark="120"
   snapInterval="1"
   valueLabelJsFunction="labelFunc"
   valueLabelStyleName="valueLabel"
  />
  <Style>
   .valueLabel{
    fontSize:11;
    fontWeight:bold;
   }
  </Style>
 </KoolChart>

<HcylinderGauge/> inherits from the parent classes.
