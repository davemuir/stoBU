Class Pie3DChartStyles
 
Packageeng.charts
Classpublic class Pie3DChart

Pie3D charts are 3D circular charts divided into sectors, illustrating numerical proportion.
The arc length of each sector (and consequently its central angle and area), is proportional to the quantity it represents.

Pie3D charts, Doughnut3D charts and Stacked Pie3D charts can be created by using <Pie3DChart>.

One or more Pie2DSeries have to be assigned to the series node.

The layout is as follows:

 <KoolChart backgroundColor="0xFFFFEE" cornerRadius="12" borderStyle="solid">
   <Options>
    <Caption text="Profit (Monthly)"/>
  </Options>
  <Pie3DChart showDataTips="true">
    <series>
     <Pie3DSeries field="Profit" nameFiled="Month"/>
    </series>
   </Pie3DChart>
  </KoolChart>

The signature of dataTipJsFunction is as follows: 

 function_name(seriesId, seriesName, index, xName, yName, data, values) { ... }
seriesId: Series ID
seriesName: The name of the series, which is set as the value of the displayName property.

yName: The value of the displayName property in the verticalAxis node.
data: The record of the dataset declared for creating the chart.
values: The value of the item. The value is an array and each element in the array is different depending on the chart type.
  • Bar2D(3D)Series (Bar2DSeries, Bar3DSeries) 0: X-axis value, 1: Y-axis value
  • Column2D(3D)Series (Column2DSeries, Column3DSeries) 0: X-axis value, 1: Y-axis value
  • Area2DSeries (Area2DSeries) 0: X-axis value, 1: Y-axis value
  • Bubble3DSeries (Bubble3DSeries) 0: X-axis value, 1: Y-axis value 2: radius
  • Line2DSeries (Line2DSeries) 0: X-axis value, 1: Y-axis value
  • Pie2D(3D)Series (Pie2DSeries, Pie3DSeries) 0: value, 1: percentage, 2:nameField
