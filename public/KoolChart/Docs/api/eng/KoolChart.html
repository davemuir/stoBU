Classpublic class KoolChart

<KoolChart> is the root node of the layout. It has the properties for drawing the background and frame of the chart.
You can adjust the background, margin and frame style of the chart by using the KoolChart class (<KoolChart>).

The sample layout is as follows:

 <KoolChart backgroundColor="0xFFFFEE" cornerRadius="12" borderStyle="solid" paddingRight="10" paddingBottom="5" paddingTop="5" paddingLeft="10">
   <Options>
    <Caption text="Annual Report"/>
   </Options>
   ...
   ...
   ...
 </KoolChart>
