Classpublic class CategoryAxis

The CategoryAxis class (<CategoryAxis>) is used to create the string typed axis.
If you are not able to quantify the axis label but you are able to make the groups of labels, then you can use CategoryAixs.
e.g. If you want to use department names (e.g. Management Division, Research Division, etc.) for axis labels, then CategoryAxis can be used.
Mostly, it is used for the horizontal axis in Column charts or Line charts.
