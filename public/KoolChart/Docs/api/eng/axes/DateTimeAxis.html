Class DateTimeAxisStyles
 
Packageeng.axes
Classpublic class DateTimeAxis

The DateTimeAxis class (<DateTimeAxis>) is used to represent the time-series data.
Due to the fact that consecutive data are represented in the axis of <DateTimeAxis>, the same repetitive data will be ignored.

dataInterval
Type: Number

Sets the interval between two consecutive data. e.g. If you set the value of the dataUnits property to "seconds" and the actual interval between two consecutive data is 3 seconds, then you must set the value of the dataInterval property to "4".
By setting dataInterval="4" and dataUnits="seconds", KoolChart will treat 3 seconds as the unit of data and start rendering from data of the 4th position (This property can be ignored by chart types).
