Class RadialGradientStyles
 
Packageeng.graphics
Classpublic class RadialGradient

The RadialGradient class (<RadialGradient>) lets you specify a gradual color transition in the fill color. A radial gradient defines a fill pattern that radiates out from the center of a graphical element.
A series of GradientEntry objects (<GradientEntry>) should be defined as the child nodes of the RadialGradient node (<RadialGradient>).

The layout is as follows:

  <fill>
    <RadialGradient angle="45" focalPointRatio="-0.5">
     <entries>
      <GradientEntry color="0xC5C551" ratio="0.00" alpha="0.5"/>
      <GradientEntry color="0xFEFE24" ratio="0.33" alpha="0.5"/>
      <GradientEntry color="0xECEC21" ratio="0.66" alpha="0.5"/>
     </entries>
    </RadialGradient>
  </fill>

RadialGradient is supported only in the gauge charts.
