Class LinearGradientStyles
 
Packageeng.graphics
Classpublic class LinearGradient

The LinearGradient class (<LinearGradient>) lets you specify the fill of a graphical element, where a gradient specifies a gradual color transition in the fill color.
A series of GradientEntry objects (<GradientEntry>) should be defined as the child nodes of the LinearGradient node (<LinearGradient>).

The layout is as follows:

  <fill>
    <LinearGradient angle="45">
     <entries>
      <GradientEntry color="0xC5C551" ratio="0.00" alpha="0.5"/>
      <GradientEntry color="0xFEFE24" ratio="0.33" alpha="0.5"/>
      <GradientEntry color="0xECEC21" ratio="0.66" alpha="0.5"/>
     </entries>
    </LinearGradient>
  </fill>

LinearGradient is supported only in gauge charts.
