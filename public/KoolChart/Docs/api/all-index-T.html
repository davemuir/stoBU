type — Style in class eng.charts.Area2DChart
Sets the representation type of area2D charts.
The valid values are as follows:
  • "overlaid": The data elements of each series are displayed overlaid.
  • "stacked": The data elements of each series are displayed as the vertical stack.
  • "100%": The data elements of each series are displayed as the vertical stack by calculating the percentage of the series sum.
