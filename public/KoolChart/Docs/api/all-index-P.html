padding — Style in class eng.axes.CategoryAxis

Sets the margin.
The default value of the padding property of Bar charts and Column charts is 0.5.
If the value of the padding property of Line charts and Area charts is 0, then there will be no margin.
The unit is the size of the space for displaying a single item in the series.
