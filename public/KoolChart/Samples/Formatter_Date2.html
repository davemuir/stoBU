This sample is to display date as the format of D MMM, YY. 
DateFormatter
formatString: For example, if you want to display date as 2009/12/01, 
you need to define the formatter as "YYYY/MM/DD".
	┌──────────────────────────────┐
	│ Y │ Year  │ YY=05,YYYY=2013  │ 
	│ M │ Month │ M=7,MM=07,MMM=Jul│
	│ D │ Day   │ D=4,DD=05	       │
	│ H │ Hour  │ H=1,HH=01	       │
	│ N │ Minute│ N=1,NN=01	       │
	│ S │ Second│ SS=30            │
	└──────────────────────────────┘

●Layout
<DateFormatter id="dateFmt" formatString="D MMM, YY"/>
